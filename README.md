# Image Reader
OCR - Reconhecimento de caracteres a partir de imagens ou arquivo PDF.

## Requisitos mínimos do sistema
* Windows 7 SP1
* Processador de 1 GHz
* 512 MB de memória RAM
* 4.5 GB de espaço disponível em disco
* Monitor com resolução HD (1280x720)

### Configurações iniciais de desenvolvimento
* Language: C#
* IDE: Visual Studio Community 2017 (15.2.26430.6) with Microsoft .NET Framework (4.7.02046)
